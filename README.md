# Employee Data Analysis using MySQL

This project demonstrates SQL-based data analysis using a dataset of 1000 employee records.

- Aggregate Functions

- Salary analysis
- Performance rating analysis
- Ranking employees by salary
- Percentage contribution analysis
- Date-based employee analysis
